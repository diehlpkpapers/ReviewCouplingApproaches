# Python scripts for some of the reviewed coupling approaches

## Usage

```bash
virtualenv deps
source deps/bin/activate
pip install -r requirements.txt
make 
```

### Scripts

* paper1.py [1]
* paper4.py [2]
* paper5.py [3]
